# UnityDevelopment_Clue-Game
Welcome to my CLUE! game project. This repo holds everything I built for my university Software Engineering module, where the goal was to design and implement a fully functioning digital version of the classic detective board game.
The project demonstrates core engineering principles including modular design, object‑oriented architecture, version control workflows, and iterative development practices.

The game recreates the classic deduction experience: players explore rooms, collect clues, and reason about suspects, weapons, and locations to solve the mystery. The codebase is structured to highlight maintainability, testability, and clear separation of concerns, reflecting industry‑aligned development standards.

Features include:

Turn‑based gameplay logic

Player movement and board navigation

Evidence tracking and deduction mechanics

Robust class structure and documentation

Unit tests and validation for key components

This project showcases both technical implementation and the engineering process behind it.
